# 예산 맞춤 섹터별 종목 추천 (데모)

예산과 관심 섹터 비중을 입력하면 섹터별 대표 종목과 매수 수량을 계산해서 보여주는 Streamlit 데모 앱입니다.

> ⚠️ **이 앱은 투자 자문이 아닙니다.** 참고용 데모이며, 실제 투자 판단과 결과에 대한 책임은 전적으로 사용자 본인에게 있습니다.

## 1. 설치

```bash
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## 2. KRX 계정 준비 (실 시세를 쓰려면 필요)

이 앱은 시세 데이터를 [pykrx](https://github.com/sharebook-kr/pykrx)로 가져옵니다. pykrx 최신 버전은
data.krx.co.kr 로그인 세션을 사용하므로, `KRX_ID`/`KRX_PW` 환경 변수가 없으면 실 시세 대신
예시(mock) 데이터로 자동 전환됩니다.

1. https://data.krx.co.kr 접속 후 무료 회원가입 (증권 계좌 개설이 아닌 사이트 로그인 ID/PW입니다)
2. 발급받은 아이디/비밀번호를 아래 `.env` 파일에 입력

## 3. `.env` 설정

`.env.example`을 복사해 `.env`를 만들고 값을 채워주세요.

```bash
cp .env.example .env
```

```
KRX_ID=발급받은_아이디
KRX_PW=발급받은_비밀번호
```

`.env`는 `.gitignore`에 포함되어 있어 커밋되지 않습니다. 계정 정보를 입력하지 않아도 앱은
정상 동작하며, 이 경우 화면 상단에 mock 데이터 사용 안내 배너가 표시됩니다.

## 4. 실행

```bash
streamlit run app.py
```

브라우저가 자동으로 열리지 않으면 터미널에 표시된 `http://localhost:8501` 주소로 접속하세요.

## 동작 방식

- 8개 섹터(반도체, 2차전지, 바이오/헬스케어, IT/플랫폼, 게임/엔터, 자동차, 금융, 화장품/소비재)별로
  큐레이션된 대표 종목 중, 시가총액 순위 점수(50%)와 최근 20영업일 모멘텀 점수(50%)를 합산해 정렬합니다.
- 선택한 섹터 비중(정규화)만큼 예산을 나누고, 섹터 안에서는 점수 비례로 예산을 재배분해
  매수 수량(`floor(배분금액 / 현재가)`)을 계산합니다. 1주도 살 수 없는 종목은 결과에서 제외됩니다.
- 시세 데이터는 10분(`ttl=600`) 캐시됩니다.
- KRX 로그인 실패, 네트워크 오류, 휴장일 등으로 실 시세를 가져오지 못한 종목은 mock 데이터로
  자동 대체되며, 화면에 대체 사유가 표시됩니다.
